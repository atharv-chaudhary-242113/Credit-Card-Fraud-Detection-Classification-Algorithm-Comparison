# Credit Card Fraud Detection: Classification Algorithm Comparison
---
## Project Overview
This project implements and compares three classification algorithms (Decision Trees, Naïve Bayes, and K-Nearest Neighbors) to detect fraudulent credit card transactions.

---
## Dataset
- **Source**: [Kaggle Credit Card Fraud Detection Dataset](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
- **Features**: 30 anonymized features (V1-V28 from PCA transformation, Time, Amount)
- **Target**: Class (0 = legitimate, 1 = fraudulent)
- **Challenge**: Highly imbalanced dataset (~0.17% fraud cases)
---
## Objective
1. Preprocess and balance the imbalanced dataset
2. Implement three classification algorithms
3. Evaluate performance using appropriate metrics
4. Compare models and recommend the best approach for fraud detection.
